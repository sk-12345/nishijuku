西塾の柔道理念

柔道をとおし、『強い心』『強い体』なにより、仲間を大切にする『思いやりの心』を
育て、チームのメンバーとの切磋琢磨で、心も体も成長するチームを目指しています。
保護者も塾生も、ともに成長できる道場！そんな道場を目指し、
『勝った時の嬉しさ』『負けた時の悔しさ』子供たちと一緒に、ともに喜び、ともに悔しがる。
保護者も子供も一緒になって世界一の道場を作って行きたいと考えています。
